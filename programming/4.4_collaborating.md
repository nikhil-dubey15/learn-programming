# 4.4 Collaborating: Sharing the Game

So far, all your "Save Games" and "Parallel Universes" exist exclusively on your laptop. If your hard drive crashes right now, all your code is gone forever!

We need to back it up to the cloud (GitHub) and learn how to share code with other developers.

## The Cloud: Remote Repositories
We call the Git folder on your laptop the **"Local"** repository. 
We call the Git folder uploaded to GitHub the **"Remote"** repository. (Usually nicknamed `origin`).

## The Collaboration Commands

### 1. `git push` (Upload)
* **What it means:** "Take my 'Save Games' from my Local laptop, and upload them to the Remote GitHub server."
* **When to use it:** At the end of the day, or whenever you finish a big piece of work.
* **How to use it:** `git push origin main` (Meaning: push my code up to the remote server, specifically into the main timeline).

### 2. `git pull` (Download Updates)
* **What it means:** "My coworker just pushed some new code to GitHub. Download it to my laptop so my Local code is up-to-date."
* **When to use it:** Every single morning before you start working! Always make sure you have the freshest code.
* **How to use it:** `git pull origin main`

### 3. The Pull Request (PR) & Merging
Remember our parallel universe (Branch) from the last lesson? 
When you are totally finished building your new feature, you want to bring that code back into the `main` working timeline. 

We do this through a **Pull Request (PR)** on GitHub!

1. You `git push` your branch to GitHub.
2. You open GitHub in your browser and click "Create Pull Request."
3. **What is a PR?** It is literally you asking your coworkers: *"Hey team, I'm requesting to PULL my new code into the main codebase. Will someone please review my work?"*
4. Your coworkers (like your mentors) read your code. They might leave comments asking you to fix a bug.
5. Once they approve it, you click the big green **"Merge"** button on GitHub.
6. The two universes smash together! Your code is now officially part of the `main` project.
